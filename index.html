Narvik Studentradio

Stoppet

Narvik Studentradio er Narviks feteste lokalradio. Vi sender musikk 24/7 på nett og på FM 94.1 Mhz. Alt innhold er Creative Commons og bla bla bla..

Kvalitet

Både HQ og LQ er tilgjengelig i OGG/Vorbis og MP3.

HQ: Stereo lyd ca 190 kb/s. For de fleste, uproblematisk via 3G eller bedre. (MP3: 192 kb/s vbr, OGG: Q6)

LQ: Mono lyd ca 64 kb/s. Primært beregnet for personer med dårlig mobilt internett/EDGE. (MP3: 64 kb/s cbr, OGG: Q3)

Mer info om lydformater og nettlesere se wikipedia om html5 lyd eller NSR's egen forskning.
